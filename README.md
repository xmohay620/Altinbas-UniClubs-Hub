# Altinbas UniClubs Hub 🎓

A full-stack web application designed for **Altinbas University** to manage and display student clubs.

## 🚀 Features
- **Club Directory:** Browse all active university clubs.
- **Admin Dashboard:** Full CRUD (Create, Read, Update, Delete) functionality to manage clubs.
- **Accessibility:** Integrated Text-to-Speech and font scaling for better user experience.
- **Live Search:** Instant filtering of clubs by name.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Altinbas Blue Theme), JavaScript.
- **Backend:** PHP 8.2.
- **Database:** MySQL.
- **Environment:** XAMPP / Apache Server.

## 📂 Installation
1. Clone the repository into your `htdocs` folder.
2. Import the `uni_clubs_db.sql` file into your phpMyAdmin.
3. Access the site via `localhost/UNICLUBS/index.php`.
